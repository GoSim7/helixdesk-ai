# Screenshot Guide

This folder is reserved for repository screenshots used in the main project README.

## Recommended files

- `01-dashboard-overview.png`
- `02-global-settings.png`
- `03-ai-presets.png`
- `04-handover-console.png`
- `05-runtime-health.png`
- `06-routing-flow.png`

## Recommended capture rules

- Use a consistent desktop width for all admin screenshots.
- Blur or remove secrets, tokens, phone numbers, email addresses, and customer data.
- Prefer English UI labels in screenshots if the repository is meant for a broader audience.
- Keep browser chrome minimal unless it helps explain deployment context.
- Export images as PNG for clarity.

## Suggested README placement

- Hero section: optional product cover or architecture image
- Mid-page: dashboard, presets, handover
- Lower page: health view or routing flow
